Flight Route Planner

Kyiv route map

OpenStreetMap in Leaflet with a direct flight path overlay between your selected points.

Quick guide

Start here
  1. Set the route with two map clicks, or type both places and the route builds automatically.
  2. Use Swap to reverse the route, and Clear to start over.
  3. When the line appears, use Preview PNG for an image and Export route ZIP for route data.
  4. Use Raster tiles, Project Rasters, and Local Raster only if you need extra map layers.

Main map: OpenStreetMap is used as the base map.

Route: a direct flight line is drawn between the selected points.

Time: the estimated flight duration is calculated locally.

Click the map: first point sets start, second point sets destination.

Flight path: the route is always shown as a direct line between both points.

Raster: switch OSM tile styles, add project rasters, or upload local overlays.
